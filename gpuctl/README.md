# gpuctl
```
Usage: gpuctl [options]<br/>
<br/>
 Options:<br/>
  -h, --help                     display this help and exit<br/>
  -l, --list-gpus                print available gpus to stdout<br/>
  -g, --gpu <busid>              check if gpu with busid is available and return EXIT_SUCCESS/EXITFAILURE<br/>
```
